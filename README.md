- iOS 16.0+
- Xcode 14.2+

## Architecture

The app works with [CleanSwift](https://clean-swift.com/) architecture

## Technologies

This app use third party libraries and native frameworks:
- UIKit
- [Nuke](https://github.com/kean/Nuke)
- Custom SPM's (Networking and Extensions)
